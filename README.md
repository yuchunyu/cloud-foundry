cloud-foundry
=============

云工厂


项目说明：本项目针对我的github上的项目进行代码拉取到服务器上运行的一个应用管理平台
初步设想一下内容
1.代码拉取
2.运行、停止、重启应用
3.日志预警信息

主要分为后台管理和前台展示两部分

所用技术 主要为nodejs
